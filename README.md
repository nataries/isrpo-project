# Приложение "Ставки на спорт"

**ФИО:** Заставная Наталия Владимировна

**Группа:** ИСП-231

## Описание

**Betting Sports** - приложение для истинных ценителей спорта! Благодаря ему вы можете поставить виртуальные деньги на победу команды-хозяина матча (командой-хозяином называется та команда, которая проводит матч на своем стадионе)

## 🕹️Как пользоваться

1. Нажмите на кнопку "Сделать ставку" на том матче, который вам интересен
2. Введите сумму (от 10 монет)
3. Подтвердите вашу ставку, деньги спишутся со счета
4. Завершите матч и введите счет
5. Получите свой выигрыш (начисляется при победе команды-хозяина)


## Какие маршруты были реализованы

|Маршрут|Описание|
|---|---|
|GET/balance|Получает текущий баланс пользователя|
|GET/matches|Получает список всех матчей|
|GET/matches/league/{league}|Получение матчей по лиге|
|POST/bet|Делает ставку на матч|
|POST/complete-match/{id}|Завершение матчей|
|POST/reset-balance|Сброс баланса до 10 000 монет|
|GET/my-bets|Получение истории всех ставок|

## Как запустить проект

### Запускаем бэкенд
1. Перейти в папку `BettingSportsApi`
2. Запустить проект через терминал командой `dotnet run`
3. Перейти по ссылке хоста

### Запускаем фронтенд
1. Открыть файл `index.html`
2. Запустить через Live Server или перенести index.html в браузер
